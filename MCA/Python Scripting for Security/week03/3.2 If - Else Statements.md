# 3.2 If - Else Statements

Conditional statements let programs make decisions based on conditions (True or False).

## Types
- `if` statement
- `if-else` statement
- Nested `if` statement
- `if-elif-else` statement

## if Statement
Executes code if the condition is True.

**Syntax:**
```python
if condition:
    # code
```

**Example:**
```python
age = 18  # Assign the value 18 to age
if age >= 18:  # Check if age is greater than or equal to 18
    print("You are eligible to vote.")  # Print the eligibility message
# Output: You are eligible to vote.
```

Multiple `if` statements check independently.

## if-else Statement
Executes one block if True, another if False.

**Syntax:**
```python
if condition:
    # code if True
else:
    # code if False
```

**Example:**
```python
age = int(input("Enter your age: "))  # Prompt user for age and convert to integer
if age >= 18:  # If age is 18 or older
    print("You are eligible to vote.")  # Print eligibility
else:  # Otherwise
    print("You are not eligible to vote.")  # Print ineligibility
# Output depends on input
```

## Nested if-else
An `if-else` inside another `if` or `else`.

**Syntax:**
```python
if condition1:
    if condition2:
        # code
    else:
        # code
else:
    # code
```

**Example:**
```python
password = input("Enter your password: ")  # Get password input from user
if len(password) >= 8:  # Check if password length is at least 8
    if any(char.isdigit() for char in password):  # Check if any character is a digit
        print("Password is strong.")  # Strong password message
    else:  # If no digit
        print("Password must contain at least one number.")  # Weak password message
else:  # If length < 8
    print("Password is too short.")  # Too short message
```

## if-elif-else Statement
Checks multiple conditions in sequence.

**Syntax:**
```python
if condition1:
    # code
elif condition2:
    # code
else:
    # code
```

**Example:**
```python
temperature = int(input("Enter the temperature in Celsius: "))  # Get temperature input
if temperature >= 30:  # If temp >= 30
    print("It's a hot day.")  # Hot message
elif temperature >= 20:  # Else if >=20
    print("The weather is warm.")  # Warm message
elif temperature >= 10:  # Else if >=10
    print("It's a cool day.")  # Cool message
else:  # Else
    print("It's a cold day.")  # Cold message
```

## Conclusion
These statements make programs dynamic by handling different scenarios.
