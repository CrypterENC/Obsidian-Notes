# Argv in Python

## What is argv?
- `sys.argv` is a list in Python that holds command-line arguments passed to the script.
- The first element (`sys.argv[0]`) is the script name.
- Subsequent elements are the arguments provided when running the script.

## Example Code
```python
import sys  # Import sys module to access argv

# Check if enough arguments are provided
if len(sys.argv) < 2:  # len(sys.argv) gives the number of arguments including script name
    print("Usage: python script.py <arg1> <arg2>")  # Print usage message if not enough args
else:
    arg1 = sys.argv[1]  # Access the first argument (index 1)
    arg2 = sys.argv[2] if len(sys.argv) > 2 else "default"  # Access second arg or use default
    print(f"Argument 1: {arg1}")  # Print the first argument
    print(f"Argument 2: {arg2}")  # Print the second argument
```

## Running the Script
- Save as `script.py`.
- Run: `python script.py hello world`
- Output: Argument 1: hello, Argument 2: world

## Notes
- Use `argparse` for more advanced argument parsing.
- Arguments are strings; convert if needed (e.g., `int(sys.argv[1])`).
