# *args and **kwargs in Python

**Key:** *args collects positional arguments into a tuple; **kwargs collects keyword arguments into a dictionary.

## *args
- Used to pass variable number of non-keyword arguments to a function.
- Arguments are collected into a tuple.
- Syntax: `def func(*args):`

## **kwargs
- Used to pass variable number of keyword arguments to a function.
- Arguments are collected into a dictionary.
- Syntax: `def func(**kwargs):`

## Examples
- `def sum_all(*args): return sum(args)`
- `def print_info(**kwargs): for k, v in kwargs.items(): print(f"{k}: {v}")`

### *args Example
```python
def multiply(*args):
    result = 1
    for num in args:
        result *= num
    return result

print(multiply(2, 3, 4))  # Output: 24
```

### **kwargs Example
```python
def user_profile(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

user_profile(name="Alice", age=30, city="New York")
# Output:
# name: Alice
# age: 30
# city: New York
```

### Combining *args and **kwargs
```python
def combined_function(a, b, *args, **kwargs):
    print(f"a: {a}, b: {b}")
    print(f"args: {args}")
    print(f"kwargs: {kwargs}")

combined_function(1, 2, 3, 4, x=5, y=6)
# Output:
# a: 1, b: 2
# args: (3, 4)
# kwargs: {'x': 5, 'y': 6}
```

## Use Cases
- Flexible functions that accept varying inputs.
- Useful in decorators, wrappers, and API calls.
- When the number of positional arguments is unknown (use *args).
- When the number of keyword arguments is unknown (use **kwargs).
- For building extensible functions without modifying signatures.
- In subclassing to handle additional parameters.
- For logging or printing variable data.
