# Debugging in VSCode

## Setup
- Install Python extension from VSCode marketplace.
- Open Python file (.py).

## Basic Debugging
- Click left of line number to set breakpoint (red dot).
- Press F5 or click Run > Start Debugging.
- Select "Python File" for configuration.

## Debugger Controls
- Continue (F5): Run to next breakpoint.
- Step Over (F10): Execute current line, skip into functions.
- Step Into (F11): Enter function calls.
- Step Out (Shift+F11): Exit current function.
- Restart (Ctrl+Shift+F5): Restart debugging.
- Stop (Shift+F5): Stop debugging.

## Features
- **Variables Panel**: View local, global variables.
- **Watch Panel**: Add expressions to watch.
- **Call Stack**: See function call history.
- **Breakpoints Panel**: Manage all breakpoints.

## Tips
- Use print statements for quick checks.
- Check terminal for errors.
- Use conditional breakpoints: Right-click breakpoint > Edit Condition.
