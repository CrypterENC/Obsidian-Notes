# Importing in Python: Basic to Advanced

## Basic Importing

### Import a Module
```python
import math
print(math.pi)
```

### Import Specific Functions
```python
from math import pi, sqrt
print(pi)
print(sqrt(16))
```

### Alias Import
```python
import math as m
print(m.pi)
```

## Intermediate Importing

### Import All from Module (Not Recommended)
```python
from math import *
print(pi)
```

### Conditional Imports
```python
try:
    import some_module
except ImportError:
    some_module = None
```

### Relative Imports in Packages
- In a package, use relative imports:
```python
from . import submodule
from .. import parent_module
```

## Advanced Importing

### Dynamic Imports
```python
import importlib
module = importlib.import_module('math')
print(module.pi)
```

### Custom Import Hooks
- Implement `__import__` or use `importlib` for custom behavior.

### Import Path Manipulation
```python
import sys
sys.path.append('/path/to/module')
import my_module
```

### Lazy Imports
```python
import lazy_import
lazy_import.lazy_module('pandas')
```

### Circular Import Avoidance
- Structure code to avoid circular dependencies.
- Use import at function level if needed.

## Best Practices
- Use absolute imports over relative.
- Avoid `from module import *`.
- Keep imports at the top.
- Use virtual environments for dependencies.
