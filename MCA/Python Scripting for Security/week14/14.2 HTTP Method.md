# HTTP Methods in Python Automation

## Common HTTP Methods
- **GET**: Retrieve data from a server
- **POST**: Send data to a server to create a resource
- **PUT**: Update an existing resource on the server
- **DELETE**: Remove a resource from the server
- **PATCH**: Partially update a resource
- **HEAD**: Retrieve headers without body
- **OPTIONS**: Get allowed methods for a resource

## Using Requests Library
```python
import requests

# GET request
response = requests.get('https://api.example.com/data')
print(response.status_code)
print(response.json())

# POST request
data = {'key': 'value'}
response = requests.post('https://api.example.com/data', json=data)
print(response.status_code)

# PUT request
response = requests.put('https://api.example.com/data/1', json=data)

# DELETE request
response = requests.delete('https://api.example.com/data/1')
```

## Notes
- Always check `response.status_code` for success (200 for OK)
- Use `json=data` for sending JSON data
- Handle exceptions with try-except for network errors
- For authentication, use `auth=(username, password)` or headers
