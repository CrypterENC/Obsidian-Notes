# Authentication in Python Requests

## Basic Authentication
- Use `auth` parameter with username and password
```python
import requests
response = requests.get('https://api.example.com/data', auth=('username', 'password'))
```

## Bearer Token Authentication
- Include token in Authorization header
```python
import requests
headers = {'Authorization': 'Bearer your_token_here'}
response = requests.get('https://api.example.com/data', headers=headers)
```

## API Key Authentication
- Often in headers or query parameters
```python
import requests
# In headers
headers = {'X-API-Key': 'your_api_key'}
response = requests.get('https://api.example.com/data', headers=headers)

# In params
params = {'api_key': 'your_api_key'}
response = requests.get('https://api.example.com/data', params=params)
```

## Digest Authentication
- Similar to basic, but more secure
```python
import requests
from requests.auth import HTTPDigestAuth
response = requests.get('https://api.example.com/data', auth=HTTPDigestAuth('username', 'password'))
```

## OAuth 1.0
- Use `requests_oauthlib` library
```python
from requests_oauthlib import OAuth1

# Setup OAuth1
auth = OAuth1('client_key', 'client_secret', 'resource_owner_key', 'resource_owner_secret')
response = requests.get('https://api.example.com/data', auth=auth)
```

## OAuth 2.0
- Use `requests_oauthlib` for OAuth2
```python
from requests_oauthlib import OAuth2Session

# Setup OAuth2
client_id = 'your_client_id'
client_secret = 'your_client_secret'
auth = OAuth2Session(client_id)

# Get authorization URL
authorization_url, state = auth.authorization_url('https://api.example.com/oauth/authorize')
print('Visit:', authorization_url)

# After user authorizes, get token
```

## Notes
- Always use HTTPS for authentication
- Store credentials securely, not in code
- Handle token expiration and refresh
- Use environment variables or config files for secrets
