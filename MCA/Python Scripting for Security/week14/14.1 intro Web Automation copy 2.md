# Web Automation

- Using scripts/programs to control web browsers automatically
- Performs repetitive tasks without human intervention
- Interacts with web pages like clicking buttons, filling forms, navigating

# Why Automation

- Saves time on repetitive tasks
- Reduces human error
- Enables 24/7 operation
- Useful for testing, monitoring, and data collection

# Tools to use

- **Selenium**: Popular for browser automation and testing
- **BeautifulSoup**: For parsing HTML and extracting data
- **Requests**: For making HTTP requests
- **Scrapy**: Framework for web scraping
- **Puppeteer**: Node.js library for browser automation
