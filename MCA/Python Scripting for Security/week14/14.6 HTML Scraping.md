# HTML Scraping in Python

## Introduction
- Extract data from HTML pages using Python
- Use BeautifulSoup to parse HTML
- Combine with requests to fetch pages

## Installation
- `pip install beautifulsoup4 requests`

## Basic Code
```python
import requests
from bs4 import BeautifulSoup

url = 'https://example.com'
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

title = soup.find('title').text
print(f'Title: {title}')

links = soup.find_all('a')
for link in links:
    print(link.get('href'))
```

## Key Methods
- `soup.find('tag')`: Find first element
- `soup.find_all('tag')`: Find all elements
- `element.text`: Get text content
- `element.get('attr')`: Get attribute

## Notes
- Check status: `response.status_code`
- Use headers to avoid blocks
