## Question 2: Student Class
Create a Student class with:

**Class attributes:**
* school_name = "Python High"

**Instance attributes:**
* name
* grade
* score

**Methods:**
* passed() that returns True if score ‚â• 60, else False

Create 5 students and print how many passed.

## Answer for Question 2
```python
# Define the Student class
class Student:
    # Constructor to initialize instance attributes
    def __init__(self, name, grade, score):
        # Set the name attribute
        self.name = name
        # Set the grade attribute
        self.grade = grade
        # Set the score attribute
        self.score = score
    
    # Method to check if student passed
    def passed(self):
        # If score is 60 or above
        if self.score >= 60:
            # Print congratulations message
            print(f"Hello {self.name}, Congrts You have passed your examinations with {self.score}")
            # Return True
            return True
        else:
            # Print failure message
            print(f"Hello {self.name}, Sorry You have Failed your examinations with {self.score}")
            # Return False
            return False

# Create a list of student objects
students = [
    Student("Alice", "A", 85),
    Student("Nittin", "C+", 55),
    Student("Nobby", "A+", 95),
    Student("Susan", "B+", 85),
    Student("Nihal", "B", 75)
]

# Initialize counter for passed students
passed_stud = 0
# Loop through each student
for i in students:
    # If the student passed, increment counter
    if i.passed():
        passed_stud += 1

# Print the number of passed students
print(passed_stud)
```

## Question 3: Bank Account
Create a BankAccount class with:

**Instance attributes:**
* owner
* balance (default 0)

**Methods:**
* deposit(amount)
* withdraw(amount)
* check_balance()

Prevent withdrawal if insufficient funds (print error)

Create an account, deposit $100, withdraw $30, check balance.

## Answer for Question 3
```python
# Define the BankAccount class
class BankAccount:
    # Constructor with default balance
    def __init__(self, owner, balance=0):
        # Set owner attribute
        self.owner = owner
        # Set balance attribute
        self.balance = balance
    
    # Method to deposit money
    def deposit(self, amount):
        # Add amount to balance
        self.balance += amount
    
    # Method to withdraw money
    def withdraw(self, amount):
        # Check if balance is sufficient
        if self.balance >= amount:
            # Subtract amount from balance
            self.balance -= amount
        else:
            # Print error message
            print("Insufficient funds")
    
    # Method to check balance
    def check_balance(self):
        # Print current balance
        print(f"Balance: ${self.balance}")

# Create a bank account
account = BankAccount("John")
# Deposit $100
account.deposit(100)
# Withdraw $30
account.withdraw(30)
# Check balance
account.check_balance()
```

## Question 4: Rectangle Class
Create a Rectangle class with:

**Instance attributes:**
* length
* width

**Methods:**
* area() returns length √ó width
* perimeter() returns 2 √ó (length + width)

Create two rectangles and compare which has larger area.

## Answer for Question 4
```python
# Define the Rectangle class
class Rectangle:
    # Constructor to initialize length and width
    def __init__(self, length, width):
        # Set length attribute
        self.length = length
        # Set width attribute
        self.width = width
    
    # Method to calculate area
    def area(self):
        # Return length times width
        return self.length * self.width
    
    # Method to calculate perimeter
    def perimeter(self):
        # Return 2 times (length + width)
        return 2 * (self.length + self.width)

# Create first rectangle
rect1 = Rectangle(5, 3)
# Create second rectangle
rect2 = Rectangle(4, 6)

# Compare areas and print result
if rect1.area() > rect2.area():
    print("First rectangle has larger area")
elif rect2.area() > rect1.area():
    print("Second rectangle has larger area")
else:
    print("Both have the same area")
```

## Question 7: Circle Class
Create a Circle class with:

**Class attributes:**
* pi = 3.14159

**Instance attributes:**
* radius

**Methods:**
* area() returns œÄ √ó r¬≤
* circumference() returns 2 √ó œÄ √ó r

Create a circle with radius 5 and print its area.

## Answer for Question 7
```python
# Define the Circle class
class Circle:
    # Class attribute for pi
    pi = 3.14159
    
    # Constructor
    def __init__(self, radius):
        # Set radius attribute
        self.radius = radius
    
    # Method to calculate area
    def area(self):
        # Return pi * radius squared
        return Circle.pi * self.radius ** 2
    
    # Method to calculate circumference
    def circumference(self):
        # Return 2 * pi * radius
        return 2 * Circle.pi * self.radius

# Create a circle with radius 5
circle = Circle(5)
# Print the area
print(f"Area: {circle.area()}")
```

## Question 8: Employee Class
Create an Employee class with:

**Instance attributes:**
* name
* hourly_rate
* hours_worked

**Methods:**
* salary() returns hourly_rate √ó hours_worked
* raise_salary(percent) increases hourly_rate by given percent

Create an employee, calculate salary, give 10% raise, calculate new salary.

## Answer for Question 8
```python
# Define the Employee class
class Employee:
    # Constructor
    def __init__(self, name, hourly_rate, hours_worked):
        # Set name attribute
        self.name = name
        # Set hourly_rate attribute
        self.hourly_rate = hourly_rate
        # Set hours_worked attribute
        self.hours_worked = hours_worked
    
    # Method to calculate salary
    def salary(self):
        # Return hourly_rate * hours_worked
        return self.hourly_rate * self.hours_worked
    
    # Method to raise salary
    def raise_salary(self, percent):
        # Increase hourly_rate by percent
        self.hourly_rate *= (1 + percent / 100)

# Create an employee
emp = Employee("John", 20, 40)
# Print initial salary
print(f"Salary: ${emp.salary()}")
# Give 10% raise
emp.raise_salary(10)
# Print new salary
print(f"New salary: ${emp.salary()}")
```

## Question 9: Smartphone Class
Create a Smartphone class with:

**Instance attributes:**
* brand
* model
* battery_level (default 100)

**Methods:**
* call(minutes) reduces battery by 2 per minute
* charge(amount) increases battery (max 100)
* check_battery() prints "Battery: {level}%"

Create a phone, call for 10 minutes, charge for 5%, check battery.

## Answer for Question 9
```python
# Define the Smartphone class
class Smartphone:
    # Constructor with default battery
    def __init__(self, brand, model, battery_level=100):
        # Set brand attribute
        self.brand = brand
        # Set model attribute
        self.model = model
        # Set battery_level attribute
        self.battery_level = battery_level
    
    # Method to make a call
    def call(self, minutes):
        # Decrease battery by 2 per minute
        self.battery_level -= 2 * minutes
    
    # Method to charge
    def charge(self, amount):
        # Increase battery, max 100
        self.battery_level = min(100, self.battery_level + amount)
    
    # Method to check battery
    def check_battery(self):
        # Print battery level
        print(f"Battery: {self.battery_level}%")

# Create a smartphone
phone = Smartphone("Samsung", "Galaxy")
# Make a 10-minute call
phone.call(10)
# Charge by 5
phone.charge(5)
# Check battery
phone.check_battery()
```

## Question 10: Library System
Create a LibraryItem class with:

**Instance attributes:**
* title
* checked_out (default False)

**Methods:**
* check_out() sets checked_out to True
* return_item() sets checked_out to False
* is_available() returns True if not checked_out

Create a book and DVD, check out the book, print availability of both.

## Answer for Question 10
```python
# Define the LibraryItem class
class LibraryItem:
    # Constructor
    def __init__(self, title, checked_out=False):
        # Set title attribute
        self.title = title
        # Set checked_out attribute
        self.checked_out = checked_out
    
    # Method to check out
    def check_out(self):
        # Set checked_out to True
        self.checked_out = True
    
    # Method to return
    def return_item(self):
        # Set checked_out to False
        self.checked_out = False
    
    # Method to check availability
    def is_available(self):
        # Return not checked_out
        return not self.checked_out

# Create book and DVD
book = LibraryItem("Harry Potter")
dvd = LibraryItem("Inception")

# Check out the book
book.check_out()

# Print availability
print(f"Book available: {book.is_available()}")
print(f"DVD available: {dvd.is_available()}")
```
