****
Lists are ordered, mutable collections that can hold items of different types, including duplicates.

## Creating Lists
```python
empty = []
numbers = [1, 2, 3]
mixed = [1, 'hello', True, [4, 5]]
```

## Characteristics
- **Ordered**: Elements keep insertion order.
- **Mutable**: Can modify after creation.
- **Heterogeneous**: Different data types allowed.
- **Duplicates**: Allowed.

## Accessing Elements
- Index starts at 0.
- Negative index for end: `list[-1]`

**Example:**
```python
lst = [10, 20, 30]
print(lst[0])  # 10
print(lst[-1])  # 30
```

## Modifying Lists

### Adding Elements
- `append(item)`: Add to end.
- `insert(index, item)`: Add at index.
- `extend(iterable)`: Add multiple from iterable.

**Example:**
```python
lst = [1, 2]
lst.append(3)      # [1, 2, 3]
lst.insert(1, 4)   # [1, 4, 2, 3]
lst.extend([5, 6]) # [1, 4, 2, 3, 5, 6]
```

### Updating Elements
Use indexing: `lst[index] = new_value`

**Example:**
```python
lst[1] = 99  # Changes second element
```

### Removing Elements
- `remove(item)`: Remove first occurrence.
- `pop(index)`: Remove and return at index (default last).
- `clear()`: Empty list.

**Example:**
```python
lst = [1, 2, 3, 2]
lst.remove(2)     # [1, 3, 2]
popped = lst.pop()  # popped=2, lst=[1, 3]
lst.clear()       # []
```

## Iterating Over Lists
```python
for item in lst:
    print(item)
```

## Nested Lists
Lists can contain other lists.

**Example:**
```python
matrix = [[1, 2], [3, 4]]
print(matrix[0][1])  # [index][value] ==> [0][1] ==> 2
```

## Built-in Functions
- `len(lst)`: Number of elements
- `min(lst)`, `max(lst)`, `sum(lst)`: Min, max, sum
- `sorted(lst)`: Return new sorted list
- `lst.sort()`: Sort in place, returns None
- `lst.reverse()`: Reverse in place

**Example:**
```python
lst = [3, 1, 4]
print(len(lst))      # 3
print(sorted(lst))   # [1, 3, 4] (new list)
lst.sort()           # lst becomes [1, 3, 4]
print(sum(lst))      # 8
```

## Python List Methods

| Method       | Description                |
| ------------ | -------------------------- |
| append(x)    | Add x to end               |
| extend(iter) | Add all from iter          |
| insert(i, x) | Insert x at i              |
| remove(x)    | Remove first x             |
| pop(i)       | Remove at i (default last) |
| clear()      | Empty list                 |
| index(x)     | First index of x           |
| count(x)     | Number of x                |
| sort()       | Sort in place              |
| reverse()    | Reverse in place           |
| copy()       | Shallow copy               |

**Examples:**
- `lst.append(4)`: Add 4 to end
- `lst.remove(2)`: Remove first 2
- `print(lst.index(3))`: Find index of 3
- `lst.sort()`: Sort list

**Syntax & Brief Example for Each:**

- **append(x)**: `lst.append(x)` ? Adds x to end. Example: `[1,2].append(3)` ? `[1,2,3]`
- **extend(iter)**: `lst.extend(iter)` ? Adds elements from iter. Example: `[1,2].extend([3,4])` ? `[1,2,3,4]`
- **insert(i, x)**: `lst.insert(i, x)` ? Inserts x at index i. Example: `[1,3].insert(1,2)` ? `[1,2,3]`
- **remove(x)**: `lst.remove(x)` ? Removes first x. Example: `[1,2,2].remove(2)` ? `[1,2]`
- **pop(i)**: `lst.pop(i)` ? Removes and returns at i. Example: `[1,2,3].pop(1)` ? 2, list `[1,3]`
- **clear()**: `lst.clear()` ? Empties list. Example: `[1,2].clear()` ? `[]`
- **index(x)**: `lst.index(x)` ? Returns index of x. Example: `[1,2,3].index(2)` ? 1
- **count(x)**: `lst.count(x)` ? Counts occurrences. Example: `[1,2,2].count(2)` ? 2
- **sort()**: `lst.sort()` ? Sorts in place. Example: `[3,1].sort()` ? `[1,3]`
- **reverse()**: `lst.reverse()` ? Reverses in place. Example: `[1,2,3].reverse()` ? `[3,2,1]`
- **copy()**: `lst.copy()` ? Returns shallow copy. Example: `[1,2].copy()` ? `[1,2]` (new list)
