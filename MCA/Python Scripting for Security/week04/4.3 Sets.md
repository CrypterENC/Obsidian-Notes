
Sets are unordered collections of unique elements. Mutable, no duplicates, unindexed.

## Creating Sets
- Curly braces: `{1, 2, 3}`
- set() function: `set([1, 2, 3])`
- Empty: `set()`

**Examples:**
```python
int_set = {1, 2, 3}
str_set = {'a', 'b'}
mixed = {1, 'a', 2.5}
```

## Characteristics
- Unordered: No order maintained.
- Unindexed: Can't access by index.
- Unique: No duplicates.
- Mutable: Can add/remove elements.
- Elements must be immutable: Sets can only contain hashable (immutable) datatypes like numbers, strings, tuples, and frozensets. Lists and other sets cannot be elements.

**Example:**
```python
# Valid elements
s = {1, 'hello', (1, 2)}  # OK

# Invalid: Lists are mutable
# s = {[1, 2]}  # TypeError: unhashable type: 'list'
```

## Accessing Elements
Iterate with loops:
```python
for item in my_set:
    print(item)
```

## Modifying Sets

### Adding
- `add(item)`: Single element.
- `update(iterable)`: Multiple.

**Examples:**
```python
s = {1, 2}
s.add(3)        # {1, 2, 3}
s.update([4,5]) # {1, 2, 3, 4, 5}
```

### Removing
- `remove(item)`: Removes item, raises error if not found.
- `discard(item)`: Removes item, no error if not found.
- `pop()`: Removes random element.
- `clear()`: Empties set.

**Examples:**
```python
s = {1, 2, 3, 4, 5}
print("Original set:", s)

# remove(item): Removes specified item, raises KeyError if not found
s.remove(3)
print("After remove(3):", s)  # {1, 2, 4, 5}

# discard(item): Removes specified item, no error if not found
s.discard(6)  # No error, set unchanged
print("After discard(6):", s)  # {1, 2, 4, 5}

# pop(): Removes and returns a random element
removed = s.pop()
print("Popped:", removed, "Set now:", s)  # e.g., Popped: 1 Set now: {2, 4, 5}

# clear(): Empties the set
s.clear()
print("After clear():", s)  # set()
```

## Set Operations
- **Union**: Combines all unique elements from both sets (no duplicates). Like merging two sets. `A | B` or `A.union(B)`
- **Intersection**: Finds elements that are in both sets. `A & B` or `A.intersection(B)`
- **Difference**: Elements in the first set but not in the second. `A - B` or `A.difference(B)`

**Examples:**
```python
A = {1, 2, 3}
B = {2, 3, 4}

# Union: All from A and B
print(A | B)  # {1, 2, 3, 4}

# Intersection: Common in A and B
print(A & B)  # {2, 3}

# Difference: In A but not in B
print(A - B)  # {1}
```

## Set Comprehension
Creates a set using a concise expression, similar to list comprehension but produces a set (unique elements).

**Example:**
```python
# Create a set of squares for numbers 0 to 4
squares = {x**2 for x in range(5)}  # {0, 1, 4, 9, 16}
# No duplicates, even if expression repeats
```

## Frozenset
An immutable version of a set. Once created, you can't add, remove, or change elements. Useful as dictionary keys or elements in other sets because it's hashable.

**Example:**
```python
# Create a frozenset
fs = frozenset([1, 2, 3])
print(fs)  # frozenset({1, 2, 3})

# Can't modify
# fs.add(4)  # AttributeError

# Can use as dict key
d = {fs: 'value'}
print(d[fs])  # 'value'
```

## Set Methods

### Basic Methods
- **add(item)**: Adds single item if not present. `s.add(4)`
- **clear()**: Removes all items. `s.clear()`
- **copy()**: Returns shallow copy. `new_s = s.copy()`
- **discard(item)**: Removes item if present, no error. `s.discard(3)`
- **remove(item)**: Removes item, raises KeyError if not. `s.remove(3)`
- **pop()**: Removes and returns random item. `item = s.pop()`
- **update(iterable)**: Adds multiple items. `s.update([4,5])`

### Set Operation Methods
- **union(*others)**: Returns union. `s.union(t)`
- **intersection(*others)**: Returns intersection. `s.intersection(t)`
- **difference(*others)**: Returns difference. `s.difference(t)`
- **symmetric_difference(other)**: Returns symmetric difference. `s.symmetric_difference(t)`
- **intersection_update(*others)**: Updates set to intersection.
- **difference_update(*others)**: Updates set to difference.
- **symmetric_difference_update(other)**: Updates set to symmetric difference.
- **issubset(other)**: True if subset. `s.issubset(t)`
- **issuperset(other)**: True if superset. `s.issuperset(t)`
- **isdisjoint(other)**: True if no common elements. `s.isdisjoint(t)`

**Examples:**
```python
s = {1,2,3}
t = {2,3,4}

print(s.union(t))  # {1,2,3,4}
print(s.intersection(t))  # {2,3}
print(s.difference(t))  # {1}
print(s.symmetric_difference(t))  # {1,4}
```
