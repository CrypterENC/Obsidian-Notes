
Dictionaries store data as key-value pairs. Keys are unique, values can be anything. Mutable, unordered (but ordered from Python 3.7), indexed by keys.

## Creating Dictionaries
- `{}`: `d = {'a': 1, 'b': 2}`
- `dict()`: `d = dict(a=1, b=2)`
- Empty: `d = {}`

## Characteristics
- Mutable: Can add, update, remove.
- Unique keys: Duplicates overwrite.
- Heterogeneous: Any types for keys/values.
- Fast lookup by key.

## Accessing Items
- `d[key]`: Raises KeyError if not found.
- `d.get(key)`: Returns None if not found.

**Example:**
```python
d = {'name': 'Alice', 'age': 25}
print(d['name'])  # Alice
print(d.get('city'))  # None
```

## Modifying Dictionaries

### Adding/Updating
- `d[new_key] = value`: Adds if key not present, updates if present.

### Removing
- `del d[key]`: Removes key.
- `d.pop(key)`: Removes and returns value.
- `d.popitem()`: Removes and returns last (key, value).
- `d.clear()`: Empties dict.

**Examples:**
```python
d = {'a': 1, 'b': 2, 'c': 3}
del d['b']  # Removes 'b'
print(d.pop('c'))  # Removes and returns 3
print(d.popitem())  # Removes and returns ('a', 1)
d.clear()  # Empties dict
```

**Examples:**
```python
d = {'a': 1, 'b': 2}
d['c'] = 3  # Add
d['a'] = 10  # Update
del d['b']  # Remove
```

## Iterating
```python
for key in d:
    print(key, d[key])
```

## Built-in Functions
- `len(d)`: Number of items.
- `key in d`: Membership test.

**Examples:**
```python
d = {'a': 1, 'b': 2, 'c': 3}
print(len(d))  # 3
print('a' in d)  # True
print('z' in d)  # False
```

## Dictionary Methods
- `d.keys()`: View of keys.
- `d.values()`: View of values.
- `d.items()`: View of (key, value) pairs.
- `d.get(key, default)`: Get with default.
- `d.update(other)`: Merge from another dict.
- `d.copy()`: Shallow copy.

**Examples:**
```python
d = {'x': 1, 'y': 2}
print(d.keys())        # dict_keys(['x', 'y'])
print(d.values())      # dict_values([1, 2])
print(d.items())       # dict_items([('x', 1), ('y', 2)])
print(d.get('z', 0))   # 0 (default)
d.update({'z': 3})
print(d)               # {'x': 1, 'y': 2, 'z': 3}
new_d = d.copy()
print(new_d)           # {'x': 1, 'y': 2, 'z': 3}
```

### Additional Methods
- **dict.fromkeys(seq, value=None)**: Creates new dict with keys from seq, all values set to value.

  Example: `dict.fromkeys(['a', 'b'], 0)` → `{'a': 0, 'b': 0}`

- **d.setdefault(key, default=None)**: Returns d[key] if exists, else sets d[key] = default and returns default.

  Example: `d.setdefault('c', 3)` → 3 (adds if not present)

**Examples:**
```python
# fromkeys
keys = ['a', 'b', 'c']
d = dict.fromkeys(keys, 0)
print(d)  # {'a': 0, 'b': 0, 'c': 0}

# setdefault
d = {'a': 1}
val = d.setdefault('b', 2)
print(val)  # 2
print(d)    # {'a': 1, 'b': 2}
```
