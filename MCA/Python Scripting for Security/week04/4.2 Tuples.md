
Tuples are immutable sequences used to store collections of items. Unlike lists, tuples cannot be modified after creation.

## Creating Tuples
- Use parentheses `()` or just commas.
- Can hold different data types, including duplicates.

**Examples:**
```python
empty = ()
numbers = (1, 2, 3)
mixed = (1, 'hello', True, [4, 5])
single = (5,)  # Note the comma for single element
```

## Characteristics
- **Ordered**: Elements keep insertion order.
- **Immutable**: Cannot change elements after creation.
- **Heterogeneous**: Different data types allowed.
- **Hashable**: Because tuples are immutable, they are hashable and can be used as dictionary keys or elements in sets.

**Example:**
```python
# Using tuple as dictionary key
coordinates = {(10, 20): 'point A', (30, 40): 'point B'}
print(coordinates[(10, 20)])  # point A

# Using in set
my_set = {(1, 2), (3, 4)}
print((1, 2) in my_set)  # True
```

## Accessing Elements
- Indexing: `tuple[index]`
- Slicing: `tuple[start:end]`

**Example:**
```python
t = (10, 20, 30, 40)
print(t[0])      # 10
print(t[1:3])    # (20, 30)
print(t[-1])     # 40 (last)
```

## Modifying Tuples
Tuples are immutable, so direct changes aren't possible. Workarounds:

### Changing Elements
Convert to list, modify, convert back:
```python
t = (1, 2, 3)
lst = list(t)
lst[1] = 99
t = tuple(lst)  # (1, 99, 3)
```

### Adding Elements
- Convert to list, append, convert back.
- Concatenate: `t + (new,)`

**Example:**
```python
t = (1, 2)
t += (3,)  # (1, 2, 3)
```

### Deleting
- Cannot delete individual elements.
- Delete entire tuple: `del t`

## Unpacking Tuples
Assign elements to variables:
```python
t = (1, 2, 3)
a, b, c = t
# a=1, b=2, c=3
```

## Iterating Over Tuples
```python
for item in t:
    print(item)
```

## Tuple Operators
- Concatenation: `+` → `(1,2) + (3,4)` → `(1,2,3,4)`
- Repetition: `*` → `('a',) * 3` → `('a','a','a')`
- Membership: `in` → `1 in (1,2,3)` → `True`

## Built-in Functions
- `len(t)`: Number of elements
- `max(t)`, `min(t)`: Max/min value
- `sum(t)`: Sum (for numbers)
- `sorted(t)`: Returns sorted list (new)

## Tuple Methods
- `t.count(item)`: Count occurrences of item
- `t.index(item)`: First index of item

**Examples:**
```python
t = (1, 2, 2, 3)
print(t.count(2))  # 2
print(t.index(3))  # 3
```

### count() Method
Counts occurrences of an item in the tuple.

**Syntax:** `tuple_name.count(item)`

**Parameters:** `item` (required) - element to count.

**Return:** Number of times item appears.

**Examples:**
```python
tuple_of_numbers = (1, 3, 4, 5, 2, 4, 6)
print(tuple_of_numbers.count(4))  # 2

tuple_of_countries = ('India', 'USA', 'France', 'USA', 'Germany', 'India', 'India', 'Brazil', 'Japan')
print(tuple_of_countries.count('India'))  # 3

# Nested: ((4, 5), 1, (4, 5), [4, 5], (2,), 4, 5)
given_tuple = ((4, 5), 1, (4, 5), [4, 5], (2,), 4, 5)
print(given_tuple.count((4, 5)))  # 2
print(given_tuple.count([4, 5]))  # 1
```

### index() Method
Returns index of first occurrence of item.

**Syntax:** `tuple_name.index(item[, start[, end]])`

**Parameters:**
- `item` (required) - element to find.
- `start` (optional) - start index.
- `end` (optional) - end index.

**Return:** Index of first occurrence. Raises ValueError if not found.

**Examples:**
```python
tuple_of_fruits = ('orange', 'apple', 'banana', 'mango', 'apple', 'melon', 'cherries', 'grapes')
print(tuple_of_fruits.index('apple'))  # 1
print(tuple_of_fruits.index('apple', 3))  # 4

# Not found: tpl_1.index(8)  # ValueError
```

## FAQs
1. **Do tuples have methods in Python?** Yes, `count()` and `index()`.

2. **Why only a few methods?** Tuples are immutable, so no modifying methods like `append()`.

3. **Can we add/remove items?** No directly. Use concatenation or convert to list.

   **Example:**
   ```python
   tpl_1 = (1, 3, 5, 2)
   tpl_2 = tpl_1 + (4,)  # Add
   print(tpl_2)  # (1, 3, 5, 2, 4)
   ```

4. **How to modify tuple?** Convert to list, modify, convert back.

   **Example:**
   ```python
   tpl_1 = (1, 3, 5, 2)
   lst_1 = list(tpl_1)
   lst_1.append(4)
   tpl_1 = tuple(lst_1)
   print(tpl_1)  # (1, 3, 5, 2, 4)
   ```

5. **Can tuples hold mutable elements?** Yes, but tuple itself is immutable.

   **Example:**
   ```python
   tpl_1 = ([1, 3], 5)
   tpl_1[0].append(2)  # Modifies list inside
   print(tpl_1)  # ([1, 3, 2], 5)
   ```
