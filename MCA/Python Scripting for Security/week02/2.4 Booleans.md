
### **What is Booleans in Python ?**
- **Python Boolean** is one of the built-in data types. 
- It is used to **represent the _truth value of an expression_.** 
	- For example, the expression **3 <= 5** is **True**, while the expression **2 == 4** is **False.**

#### **Python Boolean Type**
- A Boolean type in [Python](https://www.tpointtech.com/python-tutorial) has only two possible values:
	- **True**
	- **False**
```python
x = True  # Assign the boolean value True to variable x
y = False  # Assign the boolean value False to variable y
print(x, "->", type(x))  # Print x, "->", and the type of x
print(y, "->", type(y))  # Print y, "->", and the type of y
# Output:
# True -> <class 'bool'>
# False -> <class 'bool'>
```

> When do we Bool ? 
> 	We use it when we want to know that some [Value] is True
### **How do we Evaluating Variables and Expressions in python**.
- We can use **bool()** function, this function allows us to convert a value to a Boolean value.

**Syntax:**

```python
bool(value)
```

**Examples for `bool()` :**

```python
## **Basic Boolean values**
print(bool(True))    # Convert True to boolean and print it
print(bool(False))   # Convert False to boolean and print it

## **Numbers**

# Zero is False, non-zero is True
print(bool(0))       # Convert 0 to boolean and print it
print(bool(1))       # Convert 1 to boolean and print it
print(bool(-5))      # Convert -5 to boolean and print it
print(bool(3.14))    # Convert 3.14 to boolean and print it
print(bool(0.0))     # Convert 0.0 to boolean and print it

## Strings

# Empty string is False, non-empty is True
print(bool(""))      # Convert empty string to boolean and print it
print(bool(" "))     # Convert space string to boolean and print it
print(bool("Hello")) # Convert "Hello" to boolean and print it

## collections

# Empty collections are False, non-empty are True
print(bool([]))      # Convert empty list to boolean and print it
print(bool([1, 2]))  # Convert list [1, 2] to boolean and print it
print(bool(()))      # Convert empty tuple to boolean and print it
print(bool((1,)))    # Convert tuple (1,) to boolean and print it
print(bool({}))      # Convert empty dict to boolean and print it
print(bool({1: "a"}))# Convert dict {1: "a"} to boolean and print it
print(bool(set()))   # Convert empty set to boolean and print it
print(bool({1, 2}))  # Convert set {1, 2} to boolean and print it

## None

print(bool(None))    # Convert None to boolean and print it
```

### **Boolean Operators in Python**

Boolean operators perform logical operations on expressions, returning True or False.

#### **AND Operator**
Returns True if both operands are True.

| A     | B     | A and B |
|-------|-------|---------|
| True  | True  | True    |
| True  | False | False   |
| False | True  | False   |
| False | False | False   |

**Example:**
```python
print(True and True)   # Print the result of True and True
print(True and False)  # Print the result of True and False
print(5 > 2 and -4 < 0)  # Print the result of (5 > 2) and (-4 < 0)
```

#### **OR Operator**
Returns True if at least one operand is True.

| A     | B     | A or B |
|-------|-------|--------|
| True  | True  | True   |
| True  | False | True   |
| False | True  | True   |
| False | False | False  |

**Example:**
```python
print(True or False)   # Print the result of True or False
print(False or False)  # Print the result of False or False
print(-4 > 2 or -7 < 0)  # Print the result of (-4 > 2) or (-7 < 0)
```

#### **NOT Operator**
Reverses the Boolean value.

| A     | not A |
|-------|-------|
| True  | False |
| False | True  |

**Example:**
```python
print(not True)       # Print the negation of True
print(not False)      # Print the negation of False
print(not (6 > 1))    # Print the negation of (6 > 1)
```

#### **Equality (==) and Inequality (!=) Operators**
- `==` returns True if values are equal.
- `!=` returns True if values are not equal.

**Example:**
```python
print(8 == 8)   # Print whether 8 equals 8
print(8 != 6)   # Print whether 8 is not equal to 6
print(True == 1)  # Print whether True equals 1
print(False == 0) # Print whether False equals 0
```

#### **'is' Operator**
Checks if two variables point to the same object in memory.

**Example:**
```python
a = [1, 2, 3]  # Assign list [1, 2, 3] to variable a
b = a  # Assign the same object a to b
c = [1, 2, 3]  # Assign a new list [1, 2, 3] to c
print(a is b)  # Print whether a and b refer to the same object
print(a is c)  # Print whether a and c refer to the same object
```

#### **'in' Operator**
Checks if a value exists in a container (list, set, string, etc.).

**Example:**
```python
int_list = [12, 17, 23]  # Assign list [12, 17, 23] to int_list
fruits = {'apple', 'plum'}  # Assign set {'apple', 'plum'} to fruits
sample_str = 'tpointtech'  # Assign string 'tpointtech' to sample_str
print(17 in int_list)  # Print whether 17 is in int_list
print('apple' in fruits)  # Print whether 'apple' is in fruits
print('t' in sample_str)  # Print whether 't' is in sample_str
print('q' in sample_str)  # Print whether 'q' is in sample_str
```
