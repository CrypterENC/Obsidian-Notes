# Python Strings

**A string is a sequence of characters**, e.g., "welcome". Python has no character type; single characters are strings of length 1.

## Creating Strings
Use single or double quotes.

```python
str1 = 'Hello'  # Assign the string 'Hello' to variable str1
str2 = "World"   # Assign the string 'World' to variable str2
print(str1, str2)  # Print both str1 and str2 with a space
```

## Multiline Strings
Use triple quotes.

```python
multiline = """This is
a multiline
string."""  # Assign a multiline string to variable multiline
print(multiline)  # Print the multiline string
# Output:
# This is
# a multiline
# string.
```

## Accessing Characters
Use indexing (starts at 0).

```python
s = "Python"  # Assign the string 'Python' to variable s
print(s[0])  # Print the first character of s (index 0)
print(s[-1]) # Print the last character of s (index -1)
```

## String Slicing
Extract substrings: `s[start:end]`.

```python
s = "Python"  # Assign the string 'Python' to variable s
print(s[1:4])  # Print substring from index 1 to 3 (end exclusive)
print(s[::-1]) # Print the string reversed
```

## String Immutability
Strings cannot be changed after creation. Create new strings instead.

```python
s = "hello"  # Assign the string 'hello' to variable s
new_s = "H" + s[1:]  # Create a new string by concatenating 'H' with substring of s from index 1 to end
```

## Deleting a String
Delete the entire variable.

```python
s = "test"  # Assign the string 'test' to variable s
del s  # Delete the variable s
# s is now undefined
```

## Updating a String
Create a new string.

```python
s = "hello world"  # Assign the string 'hello world' to variable s
new_s = s.replace("world", "Python")  # Replace 'world' with 'Python' in s and assign to new_s
```

## Common Methods
- `len(s)`: Length of string.
- `s.upper()`: Uppercase.
- `s.lower()`: Lowercase.
- `s.strip()`: Remove leading/trailing spaces.
- `s.replace(old, new)`: Replace substrings.
- `s.find(sub)`: Find index of substring.
- `s.endswith(suffix)`: Check if ends with suffix.

```python
s = "  Hello  "  # Assign the string '  Hello  ' to variable s
print(len(s))  # Print the length of s (number of characters)
print(s.upper())  # Print s converted to uppercase
print(s.strip())  # Print s with leading and trailing spaces removed
print(s.replace("H", "J"))  # Print s with 'H' replaced by 'J'
print(s.find("e"))  # Print the index of the first 'e' in s
print(s.endswith("lo"))  # Print whether s ends with 'lo'
```

## Concatenation and Repetition
- `+` for concatenation.
- `*` for repetition.

```python
s1 = "Hello"  # Assign the string 'Hello' to variable s1
s2 = "World"  # Assign the string 'World' to variable s2
print(s1 + " " + s2)  # Print s1 concatenated with space and s2
print(s1 * 3)  # Print s1 repeated 3 times
```

## Formatting Strings
- F-strings: `f"{var} text"`
- `format()`: `"{} text".format(var)`

```python
name = "Alice"  # Assign the string 'Alice' to variable name
print(f"Hello {name}")  # Print formatted string using f-string
print("Hello {}".format(name))  # Print formatted string using format method
```

## Basic String Operations

### Definition
Strings are sequences of characters enclosed in single or double quotes.

Examples:
```python
'hello'  # Example of a string in single quotes
"My name is Vaishnavi"  # Example of a string in double quotes
"abc123"  # Example of a string with letters and numbers
```

Use double quotes if the string contains single quotes, and vice versa.

```python
print("Hello, my name is 'John'")  # Print the string with single quotes inside double quotes
# Output: Hello, my name is 'John'
```

### Concatenation
Combine strings with `+`.

```python
a = 'Vaishnavi '  # Assign the string 'Vaishnavi ' to variable a
b = 'Gopalasamy'  # Assign the string 'Gopalasamy' to variable b
c = a + b  # Concatenate a and b and assign to c
print(c)  # Print the concatenated string c
```

### Find
Check if a character or substring exists and get its index (starting from 0). Returns -1 if not found.

```python
a = "We're learning how to find characters"  # Assign the string to variable a
print(a.find('f'))  # Print the index of the first 'f' in a
```

### Length
Get the number of characters with `len()`.

```python
a = 'hello'  # Assign the string 'hello' to variable a
print(len(a))  # Print the length of a
```

### Multiplication
Repeat strings with `*`.

```python
a = "hello"  # Assign the string 'hello' to variable a
print(a * 5)  # Print a repeated 5 times
```
