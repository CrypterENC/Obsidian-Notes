## Definition
A function is a block of statements that performs a task. It groups repeated tasks for reuse.

## Advantages
- Reusability
- Reduces code length
- Improves readability

## Types of Functions
- Built-in: Standard Python functions
- User-defined: Created by users

## Creating a Function
Use `def` keyword.

```python
def welcome():
    print("Hello, Users! Welcome to Tpoint Tech!!")
```

## Calling a Function
Call by name followed by parentheses.

```python
welcome()
# Output: Hello, Users! Welcome to Tpoint Tech!!
```

## Function Arguments
Values passed to functions.

```python
def welcome(name):
    print(f"Hello, {name}! Welcome to Tpoint Tech!!")

welcome("John")  # Hello, John! Welcome to Tpoint Tech!!
```

```python
def addition(num1, num2):
    sum = num1 + num2
    print(num1, "+", num2, "=", sum)

addition(18, 15)  # 18 + 15 = 33
```

## Types of Function Arguments
- Positional: Based on order
- Keyword: Named arguments
- Default: With default values
- Arbitrary: *args and **kwargs

### Positional Arguments
Arguments passed in order.

```python
def greet(name, age):
    print(f"Hello {name}, you are {age} years old.")

greet("Alice", 25)  # Positional
```

### Keyword Arguments
Arguments passed by name.

```python
def greet(name, age):
    print(f"Hello {name}, you are {age} years old.")

greet(name="Alice", age=25)  # Keyword
```

### Default Arguments
Parameters with default values.

```python
def greet(name, age=18):
    print(f"Hello {name}, you are {age} years old.")

greet("Alice")  # age=18
greet("Bob", 30)  # age=30
```

### Arbitrary Arguments
*args for positional, **kwargs for keyword.

```python
def greet(*names, **details):
    for name in names:
        print(f"Hello {name}")
    for key, value in details.items():
        print(f"{key}: {value}")

greet("Alice", "Bob", age=25, city="NYC")
```

## Return Statement
Returns a value from the function.

```python
def squareOf(num):
    res = num ** 2
    return res

sqr = squareOf(13)
print("Square of 13:", sqr)  # Square of 13: 169
```

Note: Ends the function.

## Function Scope

In Python, variables have scope: where they can be accessed.

- Local: Variables defined inside a function.
- Global: Variables defined outside functions.
- To modify a global variable inside a function, use `global` keyword.
- For nested functions, use `nonlocal`.

```python
x = 10  # global

def func():
    global x
    x = 20

func()
print(x)  # 20
```

### Namespaces and LEGB Rule

Namespaces are like boxes that store names and the objects they refer to.

The LEGB rule is how Python finds a name when you use it. Python searches in this order:

1. Local: Inside the current function.
2. Enclosing: In the function that contains this one (for nested functions).
3. Global: Outside all functions, at the module level.
4. Built-in: Python's pre-defined names like `print` or `len`.

Example:

```python
x = "global"  # Global level

def outer():
    x = "enclosing"  # Enclosing level
    def inner():
        x = "local"  # Local level
        print("Inside inner:", x)  # Prints 'local'
    inner()
    print("Inside outer:", x)  # Prints 'enclosing'

outer()
print("Outside:", x)  # Prints 'global'
```
