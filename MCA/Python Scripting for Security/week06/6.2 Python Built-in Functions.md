import builtins

print(dir(builtins))

Built-in functions are pre-defined in Python and always available.

## Key Built-in Functions

### abs()
The abs() function returns the absolute value of a number, which is its distance from zero, always non-negative.

```python
print(abs(-20))  # 20
print(abs(-20.83))  # 20.83
```

### all()
The all() function returns True if all elements in the iterable are truthy, or if the iterable is empty. Otherwise, it returns False.

```python
print(all([1, 3, 4, 6]))  # True
print(all([0, False]))  # False
print(all([]))  # True
```

### any()
The any() function returns True if at least one element in the iterable is truthy. Returns False if all are falsy or the iterable is empty.

```python
print(any([4, 3, 2, 0]))  # True
print(any([0, False]))  # False
print(any([]))  # False
```

### bin()
The bin() function converts an integer to its binary string representation, prefixed with '0b'.

```python
print(bin(10))  # 0b1010
```

### bool()
The bool() function converts a value to a Boolean: True for truthy values, False for falsy ones.

```python
print(bool([]))  # False
print(bool([0]))  # True
print(bool(0.0))  # False
```

### bytes()
The bytes() function creates a bytes object from an iterable of integers or a string with encoding.

```python
array = bytes('Hello World.', 'utf-8')
print(array)  # b'Hello World.'
```

### callable()
The callable() function checks if an object can be called like a function. Returns True if callable, False otherwise.

```python
print(callable(8))  # False
```

### chr()
The chr() function returns the character corresponding to the Unicode code point integer.

```python
print(chr(97))  # 'a'
```

### compile()
The compile() function compiles source code into a code object that can be executed by exec() or eval().

```python
code = compile('x=5\ny=10\nprint("sum =",x+y)', 'sum.py', 'exec')
exec(code)  # sum = 15
```

### complex()
The complex() function creates a complex number from real and imaginary parts.

```python
print(complex(1, 2))  # (1+2j)
```

### dict()
The dict() function creates a new dictionary from keyword arguments or an iterable of key-value pairs.

```python
print(dict(a=1, b=2))  # {'a': 1, 'b': 2}
```

### divmod()
The divmod() function returns a tuple containing the quotient and remainder from dividing two numbers.

```python
print(divmod(10, 2))  # (5, 0)
```

### enumerate()
The enumerate() function adds a counter to an iterable and returns an enumerate object with index-value pairs.

```python
print(list(enumerate([1,2,3])))  # [(0, 1), (1, 2), (2, 3)]
```

### eval()
The eval() function parses and evaluates a string as a Python expression.

```python
print(eval('x + 1', {'x': 8}))  # 9
```

### exec()
The exec() function executes a string as Python code.

```python
exec('print(x==8)', {'x': 8})  # True
```

### filter()
The filter() function filters elements from an iterable using a function that tests each element.

```python
result = filter(lambda x: x > 5, [1,2,6])
print(list(result))  # [6]
```

### float()
The float() function converts a number or string to a floating-point number.

```python
print(float(9))  # 9.0
print(float("24.27"))  # 24.27
```

### format()
The format() function formats a value into a specified string format.

```python
print(format(123, "d"))  # 123
print(format(123.4567898, "f"))  # 123.456790
```

### frozenset()
The frozenset() function creates an immutable set from an iterable.

```python
fSet = frozenset(['m', 'r', 'o', 't', 's'])
print(fSet)  # frozenset({'o', 'm', 's', 'r', 't'})
```

### getattr()
The getattr() function gets the value of a named attribute from an object, with an optional default.

```python
class Details:
    age = 22

details = Details()
print(getattr(details, "age"))  # 22
```

### globals()
The globals() function returns a dictionary of the current global symbol table.

```python
print(type(globals()))  # <class 'dict'>
```

### hasattr()
The hasattr() function checks if an object has a specified attribute.

```python
class Car:
    brand = "Ford"

print(hasattr(Car, "brand"))  # True
```

### hash()
The hash() function returns the hash value of an object.

```python
print(hash(21))  # 21
```

### help()
The help() function displays help information for objects or modules.

```python
help(print)  # Shows help for print
```

### hex()
The hex() function converts an integer to its hexadecimal string representation.

```python
print(hex(10))  # 0xa
```

### id()
The id() function returns the identity of an object, a unique integer for each object.

```python
print(id("test"))  # Some number
```

### input()
The input() function reads a line from input, converts it to a string, and returns it.

```python
val = input("Enter: ")  # Prompts user
```

### int()
The int() function converts a number or string to an integer.

```python
print(int(10.52))  # 10
print(int('10'))  # 10
```

### isinstance()
The isinstance() function checks if an object is an instance of a class or its subclass.

```python
print(isinstance(5, int))  # True
```

### issubclass()
The issubclass() function checks if a class is a subclass of another class.

```python
class A: pass
class B(A): pass
print(issubclass(B, A))  # True
```

### iter()
The iter() function returns an iterator object from an iterable.

```python
it = iter([1,2,3])
print(next(it))  # 1
```

### len()
The len() function returns the number of items in an object.

```python
print(len('Python'))  # 6
```

### list()
The list() function creates a list from an iterable.

```python
print(list('abcde'))  # ['a', 'b', 'c', 'd', 'e']
```

### locals()
The locals() function returns a dictionary of the current local symbol table.

```python
def func():
    x = 5
    print(locals())  # {'x': 5}

func()
```

### map()
The map() function applies a function to all items in an iterable and returns a map object.

```python
result = map(lambda n: n*n, [1, 2, 3])
print(list(result))  # [1, 4, 9]
```

### max()
The max() function returns the largest item in an iterable or the largest of two or more arguments.

```python
print(max([1, 2, 3]))  # 3
```

### min()
The min() function returns the smallest item in an iterable or the smallest of two or more arguments.

```python
print(min([1, 2, 3]))  # 1
```

### next()
The next() function returns the next item from an iterator.

```python
it = iter([1,2,3])
print(next(it))  # 1
```

### object()
The object() function returns a new featureless object, the base for all classes.

```python
obj = object()
print(type(obj))  # <class 'object'>
```

### oct()
The oct() function converts an integer to its octal string representation.

```python
print(oct(10))  # 0o12
```

### open()
The open() function opens a file and returns a file object.

```python
f = open("file.txt", "r")  # Opens file
```

### ord()
The ord() function returns the Unicode code point for a given character.

```python
print(ord('a'))  # 97
```

### pow()
The pow() function returns the value of x to the power of y, optionally modulo z.

```python
print(pow(4, 2))  # 16
```

### print()
The print() function prints objects to the text stream file, separated by sep and followed by end.

```python
print("Hello")  # Hello
```

### range()
The range() function returns an immutable sequence of numbers, commonly used for looping a specific number of times in for loops.

```python
print(list(range(5)))  # [0, 1, 2, 3, 4]
```

### reversed()
The reversed() function returns a reverse iterator over the values of the given sequence.

```python
print(list(reversed([1,2,3])))  # [3, 2, 1]
```

### round()
The round() function returns a floating-point number rounded to ndigits decimal places.

```python
print(round(10.8))  # 11
```

### set()
The set() function creates a new set object from an iterable.

```python
print(set([1,2,2,3]))  # {1, 2, 3}
```

### setattr()
The setattr() function sets the value of a named attribute of an object.

```python
class A:
    pass

a = A()
setattr(a, 'x', 5)
print(a.x)  # 5
```

### slice()
The slice() function returns a slice object that can be used to slice sequences.

```python
s = slice(1, 5)
print([1,2,3,4,5,6][s])  # [2, 3, 4, 5]
```

### sorted()
The sorted() function returns a new sorted list from the items in iterable.

```python
print(sorted([3,1,2]))  # [1, 2, 3]
```

### str()
The str() function returns a string version of an object.

```python
print(str(123))  # '123'
```

### sum()
The sum() function sums the items of an iterable from left to right and returns the total.

```python
print(sum([1,2,3]))  # 6
```

### super()
The super() function returns a proxy object that allows you to call methods of a parent or sibling class.

```python
# Example in class
```

### tuple()
The tuple() function creates a tuple from an iterable.

```python
print(tuple([1,2,3]))  # (1, 2, 3)
```

### type()
The type() function returns the type of an object, or creates a new type if three arguments are passed.

```python
print(type(5))  # <class 'int'>
```

### vars()
The vars() function returns the __dict__ attribute of an object.

```python
class A:
    x = 5

print(vars(A))  # {'x': 5}
```

### zip()
The zip() function takes iterables, aggregates them in a tuple, and returns an iterator of tuples.

```python
print(list(zip([1,2], ['a','b'])))  # [(1, 'a'), (2, 'b')]
```
