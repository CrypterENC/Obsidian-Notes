What is Wireshark ? 
-  It is a **Traffic Analyzing too**l and it is free and open-source.

Some uses of Wireshark:
- Help to detect anomalies, intrusions and performance issues
- Network Troubleshooting - 
	- Check packet loss, retransmissions and latency.
	- Analyze TCP Handshake (SYN, SYN-ACK, ACK).
- Security Analysis:
	- Detect suspicious traffic.
	- ARP spoofing
	- Unencrypted credentials in HTTP, FTP or Telnet.
	- Analyze SSL/TLS handshakes.
- Performance Monitoring
	- Measure response times (DNS, HTTP)
	- Use "Statistics" menu for Protocol Hierarchy, Conversation(IP pairs)
