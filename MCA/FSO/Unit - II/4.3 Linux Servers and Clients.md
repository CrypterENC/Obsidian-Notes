
## Intro to Client - Server Communications

### **Server :** 
-  Servers are ~={yellow}**computers with software installed**=~ that enables them to ~={green}**provide services to client across the network.**=~

	-  **~={yellow}Some may provide external resource=~** such as ~={green}**files, email messages or web pages to client upon request.**=~
	- **~={yellow}Other services are "RUN MAINTENANCE TASK"=~** like **~={green}log management, disk scanning.=~**

### Servers, Services, and Their Ports

- A port is a **~={yellow}reserved network resource=~** **~={green}used by a service.=~**
- Server "LISTENS" on port for a client

**Well-Known Ports:**

| PORT  | Description |
| ----- | ----------- |
| 20/21 | FTP         |
| 22    | SSH         |
| 23    | TELNET      |
| 25    | SMTP        |
| 53    | DNS         |
| 67/68 | DHCP        |
| 80    | HTTP        |
| 110   | POP3        |
| 123   | NTP         |
| 443   | HTTPS       |

### Clients
- **~={yellow}Clients are programs or applications designed=~** to **~={green}communicate with a specific type of server.=~**
	-  Clients **~={green}use a well-defined protocol to communicate=~** with the server.
	-  Web Browsers are web web clients that are used to communicate with web servers through the port 80 (HTTP)
