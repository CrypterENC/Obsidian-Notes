#### What is IP Address 

- It stands for Internet protocol address
- It is an ~={green}**identifying number**=~ that is **~={green}associated with a specific computer**=~ or a computer network.

#### 2 Versions of IP ( IPv4 32bit and IPv6 128bit )

- IPv4 is the **most widely used protocol right** now whereas IPv6 is the **next generation protocol for the internet.**

##### IPv4 
- IPv4 uses **32 binary bits** to **create a single unique address on the network.** 
- An IPv4 address is expressed by **four numbers separated by dots.** --> **172.16.254.1**
- Each **number is the decimal** (base- 10) **representation for an eight- digit binary** (base-2) number, also called an octet.
