## **What is Linux ?**

- Linux is a **~={green}operating system that was created in 1991=~**. Linux is **~={cyan}open source, fast, reliable and small.=~**
- ~={yellow}**Linux=~** is a **~={green}part of several platforms=~** and can be found on devices anywhere from wristwatches to supercomputers.

### What is Value of Linux

- Linux is often the OS of choice in the SOC. These are some of the reasons to choose Linux:
	1. Linux is open source.
	2. Linux's CLI is very powerful.
	3. The user has more control over the OS.
